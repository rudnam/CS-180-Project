## Dev

- Clone the repository
```console
git clone https://github.com/rudnam/cs180-project.git
```
- Create environment
```console
cd cs180-project/api
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

- Start flask and visit `http://localhost:5000`
```console
python3 app.py
```
